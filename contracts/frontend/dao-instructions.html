Initial Setup

1. Install MetaMask

Download and install the MetaMask browser extension from metamask.io

2. Configure Sepolia Network

Add the Sepolia testnet to your MetaMask wallet with the following configuration:

  • Network Name: Sepolia
  • RPC URL: https://sepolia.infura.io/v3/YOUR_KEY
  • Chain ID: 11155111
  • Currency Symbol: ETH
3. Get Test ETH

Obtain Sepolia test ETH from the following faucets:

QuickNode Faucet Sepolia Faucet
4. Add SGL Token

Add the SGL token (DAI for testing) to your MetaMask:

  • Token Address: 0xFF34B3d4Aee8ddCd6F9AFFFB6Fe49bD371b8a357
  • Token Symbol: DAI
  • Decimals: 18

How to Use the DAO

Creating Proposals

Any member with SGL tokens can create governance proposals. Proposals can include funding requests, parameter changes, or strategic decisions.

Voting Process

Token holders can vote on active proposals. Your voting power is proportional to your SGL token balance. Votes are recorded on the blockchain and are immutable.

Treasury Management

The DAO treasury holds community funds that can be allocated through governance proposals. All treasury transactions require community approval.

Important Notes
  • This is a testnet deployment - no real value is at stake
  • Always ensure you have enough ETH for gas fees
  • Keep your wallet backup phrase secure
  • Always verify contract addresses before interacting
